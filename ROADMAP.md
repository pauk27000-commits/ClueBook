# План развития и идеи (ROADMAP)

Здесь мы собираем все идеи по улучшению модуля QuickNotes (Блокнот). 
Вы можете свободно редактировать этот файл: удалять то, что не нравится, добавлять свои мысли и помечать галочками `[x]` то, что уже сделано.

---

## 🎨 1. Читаемость и Цвета (UI/UX)
- [x] **Умные бейджи для метаданных.** Текст дедлайнов, дат и локаций помещать в полупрозрачные темные/светлые плашки (бейджи). Это решит проблему сливающегося текста (например, красного дедлайна на красной карточке).
- [x] **Динамический контраст текста.** Автоматически делать цвет шрифта темно-графитовым для светлых карточек (желтая, розовая) и белым/кремовым для темных (фиолетовая, синяя, коричневая).
- [x] **Эффект матового стекла (Glassmorphism).** Добавить карточкам на доске легкий эффект размытия заднего фона и аккуратные рамки, чтобы они смотрелись объемнее и современнее.

## 🔗 2. Прокачка связей на доске
- [x] **Меню настройки связи (Двойной клик ЛКМ).** Заменить моментальное удаление на полноценное окно редактирования нити:
  - Выбор стиля линии: сплошная (сильная связь), пунктир (слух), точки (неподтвержденная информация).
  - Выбор цвета линии: красный (враги), зеленый (союзники), серый (нейтралитет) и т.д.
  - Настройка толщины нити.
  - Установка стрелок направления (от кого к кому идет связь).
  - Удобное поле для редактирования текста (подписи) связи прямо в этом же меню.

## ⏱ 3. Хронология, Таймлайны и Даты
- [ ] **Визуальный Таймлайн (Вертикальная ось).** Переработать вкладку "Хронология", чтобы карточки красиво цеплялись к единой центральной линии времени, формируя настоящую ленту событий.
- [ ] **Раздельный ввод даты и времени.** Добавить удобный мини-календарик (Date Picker) для выбора точной даты в пару кликов, при этом оставив текстовое поле для абстрактных сроков ("Утром", "В ближайшее время").
- [ ] **Режим виртуальной сортировки.** Сделать переключатель "Порядок: Ручной / По датам". При выборе "По датам" карточки выстраиваются хронологически только визуально, не перезаписывая ваш ручной порядок сортировки (сохраненный перетаскиванием).
- [ ] **Интеграция с Simple Calendar.** Если в мире установлен модуль календаря, автоматически подтягивать текущую игровую дату при создании нового события в хронологии.

## 🗺 4. Доска (Board Canvas)
- [ ] **Зум камеры (Приближение/Отдаление).** Добавить возможность отдалять и приближать доску с помощью колесика мыши (Mouse Wheel zoom), чтобы легко окинуть взглядом огромную схему.
- [ ] **Привязка к сетке (Snap to Grid).** Добавить переключатель (точечная сетка на фоне), при включении которого карточки будут "прилипать" к ровным линиям — рай для перфекционистов.
- [ ] **Группировка (Зоны/Контейнеры).** Возможность нарисовать на доске полупрозрачную область с названием (например, "Семья Коста") и закинуть туда карточки. Двигаешь область — двигаются все карточки внутри неё.

## ✨ 5. Качество жизни (QoL)
- [ ] **Закрепление записей (Pin).** Добавить иконку "канцелярской кнопки". Закрепленные карточки всегда висят в самом верху списка в своей вкладке.
- [ ] **Rich-text или Markdown.** Добавить базовую поддержку форматирования текста (жирный, курсив, списки) внутри больших текстовых полей карточек без необходимости писать HTML-теги.
- [ ] **Мини-виджет блокнота.** Режим, при котором окно блокнота не закрывается, а сворачивается в аккуратную маленькую панель сбоку экрана для быстрого доступа.
