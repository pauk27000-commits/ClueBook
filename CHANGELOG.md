# История изменений (Changelog)

Здесь описаны все изменения и обновления модуля ClueBook (Ежедневник).

## [1.5.0] - 2026-07-07
### Добавлено
- Добавлена возможность менять порядок карточек во вкладках списков ("Заметки", "Персонажи", "Квесты", "Хронология"). Перетаскивание реализовано через "живую сортировку" (Live Swap), как в современных планировщиках.
- Добавлен файл документации QUICKNOTES_ARCHITECTURE.md для ИИ-ассистентов и разработчиков.

### Изменено
- Окно быстрого создания (Quick Add) теперь выстраивает поля ввода вертикально с красивыми отступами вместо тесной горизонтальной строки.
- Улучшен дизайн полей ввода в всплывающих окнах: темный полупрозрачный фон, белый текст и акцентные обводки.
- Текст при редактировании светлых карточек теперь окрашивается в темный цвет для идеальной читаемости.
- Иконки всплывающего меню теперь сохраняют позицию (transform), не дергаясь при масштабировании (scale).

### Исправлено
- Полностью решены проблемы совместимости с Foundry V12-V14: заменены устаревшие API TextEditor, FormDataExtended и синтаксис принудительного удаления флагов. Теперь модуль работает без предупреждений (warnings) в консоли.
- Перетаскивание плавающей иконки больше не блокирует клики по всплывающим кнопкам создания.

## [1.4.0] - 2026-07-07
### Добавлено
- Быстрое радиальное меню вокруг кнопки-виджета. При наведении курсора появляются 4 кнопки создания различных типов записей.
- Задержка исчезновения (1.5 сек) для радиального меню, чтобы было удобно наводить мышку без случайных закрытий.

## [1.3.0] - 2026-07-07
### Исправлено
- Оптимизирован механизм привязки (Links) на доске. Исправлен баг, при котором линии не обновлялись во время перемещения.
- Добавлен корректный парсинг HTML (TextEditor) для V11 и Application V2.

## [1.2.0]
- Добавлен режим "Глобальный поиск".
- Модернизация UI вкладок (Glassmorphism).

## [1.1.0]
- Интеграция с рабочими пространствами.
- Окно настроек для кастомизации прозрачности и акцентных цветов.

## [1.0.0]
- Первый релиз.
- Архитектура на основе сверхбыстрого Application V2 для работы без зависаний.

*(Для будущих обновлений добавляйте изменения сюда с указанием новой версии)*
