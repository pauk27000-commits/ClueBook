# История изменений (Changelog)

Здесь описаны все изменения и обновления модуля ClueBook.

## [1.0.1] - Патч и новые функции доски (Крупное обновление)
**Добавлено:**
- **Удаление нитей:** Добавлена возможность удалять связи между карточками на доске по правому клику мыши (клик по нити).
- **Масштабирование на доске:** Все записи, размещенные на доске, теперь можно растягивать в ширину и высоту (потянув за правый нижний угол). Размеры сохраняются индивидуально для каждой карточки.
- **Настройка внешнего вида связей:** Во вкладке "Настройки" появились опции изменения цвета нитей и их стиля (сплошная, пунктир, точки).
- **Плавающая кнопка модуля:** Если список игроков перекрывает кнопку в левом нижнем углу, любой игрок может перетащить ее мышкой в удобное место. Позиция сохраняется в локальных настройках.

**Изменено / Удалено:**
- Вкладка "Улики" была полностью удалена из модуля по запросу для упрощения интерфейса.
- Исправлены баги позиционирования при зуме и конфликты перетаскивания с масштабированием на доске.
## [1.0.0] - Первый стабильный релиз (Initial Release)
**Добавлено:**
- Полноценная система раздельных вкладок: Заметки, Персонажи (NPC), Улики, Квесты и Хронология.
- **Интерактивная доска:** Бесконечный холст с поддержкой панорамирования (ПКМ/СКМ) и масштабирования (колесико мыши). Записи можно выносить на доску и связывать между собой нитями.
- **Два режима доступа:** Личный блокнот (только для игрока) и Общий журнал (для всех игроков). Синхронизация в реальном времени.
- **Вкладка "Настройки":** 
  - Личные настройки внешнего вида: акцентные цвета и эффект стекла (настраиваемая прозрачность).
  - Скрытие неиспользуемых вкладок (NPC, Улики и т.д.) как для личного, так и для общего блокнота.
  - Настройка цвета заметок по умолчанию при создании.
- **Глобальный поиск:** Умный поиск по всем записям всех вкладок одновременно (отдельная вкладка с лупой).
- **Умное редактирование:** Двойной клик по записи для быстрого перехода в режим редактирования, автоматическое сохранение при клике вне поля (фокус теряется — запись сохраняется).
- Архитектура на основе сверхбыстрого **Application V2** для работы без зависаний.

*(Для будущих обновлений добавляйте изменения сюда с указанием новой версии)*
